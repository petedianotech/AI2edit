# **App Name**: MotionSpeak

## Core Features:

- AI Text-to-Speech: Convert text into natural-sounding speech for video narration.
- Automatic Caption Generation: Automatically generate captions from video audio using speech recognition.
- Basic Video Editing Tools: Tools for trimming, cutting, merging, and adjusting video clips.
- Transitions and Effects: Add transitions between clips and apply various visual effects to enhance video appearance.
- Audio Editing: Adjust audio levels, add background music, and include sound effects.
- Text and Title Overlays: Add text overlays and customizable titles with various fonts and styles.
- Generative AI scene tool: The app will suggest suitable videos and B-roll based on the transcription of the user's main videos, or from user's text prompt

## Style Guidelines:

- Primary color: Deep sky blue (#32ade6) for a fresh and modern feel.
- Background color: Very light blue (#E5F3FF), near-white and unobtrusive.
- Accent color: Soft green (#8ee632) to provide contrast, particularly for actionable elements and highlights.
- Body and headline font: 'Inter' for a clean and modern look, suitable for both headlines and body text.
- Use flat, outlined icons with rounded corners to maintain a friendly and approachable aesthetic.
- Clean, intuitive layout with clear sections for each editing function.
- Subtle transitions and animations for a smooth user experience.
